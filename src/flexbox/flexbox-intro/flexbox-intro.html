What is Flexbox?

  • Arrange elements in a container
  • No floats
  • Responsive (can also combine with media queries)
  • Positioning child elements is much easier
  • Flex container:s margins do not collapse with the margins of its contents
  • Order of elements can easily be changed without editing source HTML

Browser Support

Bellow Edge has some quirks.

Concept

  • The ability to alter width and height to best fit in its containers available free space
  • Flexbox is direction-agnostic (in oppose to block which is vertical biased and inline which is horizontal biased)
  • Built for small-scale layouts while the upcomming Grid is for more large scale
